关于目录

  1. _posts文件夹存放网站的文章,assets用于存放网站的资源

  2. 网站中某些图片资源主要存放在assets文件夹中,比如配置文件_config.icarus.yml中的支付宝二维码就存放在这里,使用方式/assets/img/alipay.jpg,从assets开始写

  3. 对于_posts文件夹中的内容,md格式文件后面会渲染成单独的文章,与md文件同名的文件夹中存放的文章中所使用的图片,直接写文件名就是在这个文件夹下图片

    参考内容:hexo资源文件夹

关于其他目录

  1. hexo/scaffolds目录中,存放文章建立的模板,post.md为新建文章的模板
