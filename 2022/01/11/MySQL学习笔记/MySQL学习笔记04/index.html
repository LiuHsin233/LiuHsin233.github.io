distinct

查询过程中如果想要去除重复值,使用distinct

where

用于有条件在表中选取数据

下面这些是可以在where子句中使用的操作符

操作符 作用
= 等于
<> 不等于,有些SQL中可以写成!=
> 大于
< 小于
>= 大于等于
<= 小于等于
between 在某个范围内
like 搜索某种模式

between

指定范围

like

用来查找指定条件的语句(匹配字符串)

通配符 作用 示例
% 替换一个或者多个字符 %lon%
_ 替换一个字符 _uck
[字符表] 表示表中的任意一个字符 [lf]uck
[^字符表]或者[!字符表] 表示不在表中的任意一个字符 [!123]或者[^123]

in

允许查找的时候指定多个值

top

规定需要返回的语句条目数,不同数据库系统不一定支持top

insert into

用于在指定表格中插入一条数据

update

用于更新表中的数据

delete

用于删除表中的记录
